Py-clave and Books as API

By Silvio Lorusso, July 19, 2013 at 12:59 pm.

Gabriele Alese recently initiated the project py-clave, «a prototype implementation of a set of public RESTful APIs meant to enable clients to retrieve metadata and contents from a bunch of digital publications in EPUB2 format».

A presentation on the usefulness of APIs for books by Hugh McGuire, founder of PressBooks.
